# Day 06: Advanced Conditions (Else-If Ladder)

Today I practiced multi-option conditional flows using the else-if ladder structure to control complex decision paths.

## What I Learnt:
* **Else-If Ladder:** Learned how to evaluate multiple independent conditional statements when dealing with more than two possible outcomes.
* **Top-to-Bottom Execution:** Understood that the compiler evaluates conditions sequentially from top to bottom and stops execution as soon as it encounters the first true condition.
* **Practical Application:** Built a vehicle speed tracker logic to categorize speed thresholds and flag alerts dynamically.
