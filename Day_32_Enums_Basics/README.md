# Day 32: Introduction to Enums and Switch Case

Today I learned how to use Enums to create fixed options and combined them with Switch Cases to build a clean port scanner simulation.

## What I Learnt:
* **Enum Basics:** Enums allow us to use clear names in code while the computer reads them as numbers behind the scenes.
* **Automatic Indexing:** By default, enums assign numbers starting from 0, just like array indexing. Changing the order changes the number values.
* **Custom Numbering:** I learned how to assign custom integers to enum options, which is very useful for matching real-world data like network port numbers.
* **Switch Case Control:** Understood how switch case acts like a direct jump to a specific block of code, saving execution time compared to multiple if-else rules.

## Files in this Folder:
1. `enum_basics.c` -> Basic game state controller to test enum numbering and memory size.
2. `port_scanner_practice.c` -> A network security scanner simulation using custom enum values and switch statements.
