# Day 18: Pointers and Memory Addresses

Today I learned how pointers store RAM memory addresses and allow direct manipulation of variables in memory using operators.

## What I Learnt:
* **Address Operator (`&`):** Used the address-of operator to fetch the exact memory location of a variable in RAM.
* **Dereference Operator (`*`):** Used the dereference operator to read and modify values stored inside a memory address directly.
* **Pointer Arithmetic:** Explored pointer incrementing to see how memory addresses shift based on data type size.

## Files in this Folder:
1. `intro_to_pointers.c` -> Introductory program printing variable values, memory addresses, and pointer references.
2. `pointer_arithmetic.c` -> Program demonstrating memory address movement using pointer arithmetic operations.
3. `pointer_modification.c` -> Script showing how to update a variable value directly using its pointer reference.
