### Day 18: Pointers and Memory Addresses
* **Folder:** `Day_18_Pointers`
* **File:** `hacker_ptr.c`
* **What I Learnt:**
  * Learn about pointers which store memory address of other variables.
    > Pointer ke baare mein seekha jo doosre variables ka address save karta hai.
  * Use & symbol to find the secret home address of variable in RAM.
    > Variable ka secret address dhoondne ke liye `&` use kiya.
  * Use * symbol to go inside address and print or change the value.
    > Address ke andar ja kar value badalne ya print karne ke liye `*` ka use seekha.
