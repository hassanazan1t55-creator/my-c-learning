# Day 17: String Functions Shortcuts

Today I practiced using the `<string.h>` library functions in C to handle string copying, concatenation, length checking, and comparisons cleanly.

## What I Learnt:
* **String Length (`strlen`):** Calculated string character counts directly without manually running loops.
* **String Copy (`strcpy`):** Copied text from one character array variable into another destination array.
* **String Concatenation (`strcat`):** Combined two separate strings together into a single continuous character array.
* **String Comparison (`strcmp`):** Compared two string values using `strcmp` to build a simple password checking system.

## Files in this Folder:
1. `string_magic.c` -> A program demonstrating string copying with `strcpy` and length checking with `strlen`.
2. `string_functions.c` -> A script combining first and last name strings into a full name using `strcat`.
3. `string_compare.c` -> A basic authentication program checking user input against a saved password using `strcmp`.
