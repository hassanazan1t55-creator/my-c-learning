### Day 21: Custom Data Types (Structures)
* **Folder:** `Day_21_Structures`
* **File:** `enemy_struct.c`
* **What I Learnt:**
  * Learn how to use struct keyword to make custom boxes for data.
    > struct keyword use kar ke custom variable containers banana seekha.
  * Understand the difference between structure design and real object block.
    > Structure design aur RAM ke andar asli object block ka farq samjha.
  * Use dot operator (`.`) to access variables inside a structure box.
    > Almari ke compartments ke andar value daalney ke liye dot operator seekha.
