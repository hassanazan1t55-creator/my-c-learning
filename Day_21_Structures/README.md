# Day 21: Custom Data Types (Structures)

Today I learned how to create user-defined custom data structures using the `struct` keyword to group related variables of different data types together.

## What I Learnt:
* **Custom Data Types (`struct`):** Defined custom structures to hold different data types under a single variable entity.
* **Member Access (`.` operator):** Used the dot operator to access, initialize, and display individual structure attributes.
* **Array of Structures:** Combined structures with arrays to manage and process multiple entity records sequentially using loops.

## Files in this Folder:
1. `student_structure.c` -> Basic structure implementation defining and displaying student details.
2. `user_input_structure.c` -> Program taking custom car data input from the user using structure variables.
3. `array_of_structures.c` -> Program managing an array of structure objects to store multiple car records via loops.
