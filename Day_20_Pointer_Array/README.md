### Day 20: Pointers and Arrays Relationship
* **Folder:** `Day_20_Pointer_Array`
* **File:** `array_ptr.c`
* **What I Learnt:**
  * Learn that array name is actually a pointer to the first box [0].
    > Seekha ke array ka naam khud pehle dabbe ka pointer address hota hai.
  * Use pointer arithmetic `p++` to jump directly to the next memory block.
    > Memory ke agle block par direct jump karne ke liye `p++` ka use seekha.
  * Understand the difference between Index `marks[i]` and Direct Memory Pointer jump.
    > Indexing aur direct memory address jump ke darmiyan ka hacking logic samjha.
