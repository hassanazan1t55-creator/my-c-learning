# Day 42: Reversing a Singly Linked List

Today I mastered the ultimate iterative linked list manipulation: reversing pointer node references in-place using the three-pointer technique.

## What I Learnt:
* **The 3-Pointer Technique:** Used `prev`, `current`, and `next` pointers simultaneously to slide safely across memory locations without dropping linkages.
* **In-Place Pointer Overturning:** Changed `current->next = prev` line connections systematically to inverse sequence execution paths.
* **Dynamic Head Re-assignment:** Shifted the structural `head` anchor to the final `prev` node orientation once tracking boundaries hit `NULL`.
