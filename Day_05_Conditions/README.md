# Day 05: Conditional Statements (If-Else)

Today I practiced multi-stage decision-making control logic in C using standard conditions to evaluate variable thresholds dynamically.

## What I Learnt:
* **Conditional Logic:** Mastered how to pass logical paths using `if`, `else if`, and fallback `else` statements to create multi-tier calculations.
* **Syntax Rules:** Learned that a semicolon `;` should never follow the initialization of an `if` condition directly, as it prematurely terminates the block.
* **Value Computations:** Applied percentage math operations dynamically based on logical threshold conditions to deduct smart discount totals.

## Files in this Folder:
1. `age_checker.c` -> A basic age validation check script evaluating standard authorization status for a driving license.
2. `smart_discount_calculator.c` -> An advanced multi-condition retail application calculating dynamic percentages off total shopping bills.
