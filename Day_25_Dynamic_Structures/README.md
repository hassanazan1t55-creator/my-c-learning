### Day 25: Dynamic Structures (Structures + Malloc)
* **Folder:** `Day_25_Dynamic_Structures`
* **File:** `dynamic_struct_array.c`
* **What I Learnt:**
  * Combine custom `struct` layouts with `malloc()` to request live object blocks.
    > Custom structure ko dynamic allocation ke sath mila kar runtime storage seekha.
  * Master the syntax switch between standalone pointer Arrow (`->`) and array index Dot (`.`).
    > Pointer access ke liye arrow aur array elements ke sath dot lagane ka sahi rule samjha.
  * Allocate memory dynamically based on user requirements to build zero-waste programs.
    > Memory ko utna hi use karna seekha jitna user ko zaroorat ho taake space zaya na ho.
