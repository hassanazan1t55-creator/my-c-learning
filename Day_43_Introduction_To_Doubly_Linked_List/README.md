# Day 43: Introduction to Doubly Linked Lists

Today I learned about Doubly Linked Lists (DLL). We moved from a simple one-way linked list to a two-way path where nodes can point both forward and backward.

## What I Learnt:
* **Two-Way Node Structure:** I learned how to create a node that holds both a next pointer and a prev pointer to go back and forth.
* **Going Both Ways:** I learned how to print the list from start to end, and also how to print it backwards from the last node using the prev pointer.
* **Searching values:** I combined search logic with the new doubly linked list to find numbers inside the nodes.

## Files in this Folder:
1. `doubly_list_foundation.c` -> Setting up a basic doubly linked list with three nodes.
2. `bidirectional_traversal.c` -> Code to print the list both forward and backward.
3. `doubly_list_search.c` -> Complete practice file that handles two-way printing and searching for a value.
