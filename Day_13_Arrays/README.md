# Day 13: Introduction to Arrays

Today I practiced managing sequential data allocations by using arrays to group multiple values together under a single variable tracker name.

## What I Learnt:
* **Contiguous Memory Storage:** Learned how to declare structure containers to store continuous items of the identical data type without establishing separate independent variables.
* **Zero-Based Indexing:** Mastered tracking data nodes using standard zero-offset parameters, understanding why accessing elements starts directly from index address position `0`.
* **Dynamic Loop Integration:** Combined arrays with iterative `for` loop frameworks to automate sequential user scanning operations and output processes cleanly.

## Files in this Folder:
1. `intro_to_arrays.c` -> A foundational array initialization profile demonstrating how to assign static student score lists using explicit hardcoded indexes.
2. `lucky_numbers.c` -> An interactive script focusing on targeting specific index locations to parse structured elements out of an data array set.
3. `array_with_loop.c` -> A dynamic application combining counter loops with array parameters to safely capture and print variable commercial pricing decimals.
