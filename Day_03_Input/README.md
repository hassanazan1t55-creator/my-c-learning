# Day 03: User Input Basics

Today I learned how to interact with users by capturing live inputs from the terminal and storing them safely into memory variables.

## What I Learnt:
* **Using scanf:** Learned how to use `scanf` with format specifiers like `%d`, `%f`, and `%c` to take integer, float, and character values from the user.
* **The Ampersand (&) Sign:** Understood that the `&` symbol acts as a pointer to the specific memory address where the input data should be saved.
* **Format Stability:** Practiced clearing buffer spaces (using a leading space like `" %c"`) to safely read single characters without skipping inputs.

## Files in this Folder:
1. `input.c` -> Basic user input script capturing integer and floating-point data values.
2. `product_billing_system.c` -> A practical billing system script that takes product quantity, price, and store section coordinates from the user.
