### Day 12: Functions with Return Values
* **Folder:** `Day_12_Return_Values`
* **File:** `burger_shop.c`
* **What I Learnt:**
  * Learn how to use int instead of void to get answer back from function.
    > Function se jawab wapas lene ke liye void ki jagah int use karna seekha.
  * Learn return keyword which gives output back to main pocket.
    > Return keyword ke baare mein seekha jo jawab ko main program mein bhejta hai.
  * Make a minus function that returns the result without printf inside it.
    > Andar printf likhe bina minus ka jawab return karne wala function banaya.
