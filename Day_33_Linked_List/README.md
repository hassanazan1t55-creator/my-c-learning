# Day 33: Linked Lists & Advanced Pointer Mastery in C

Today I worked on core Data Structures and Algorithms (DSA) concepts, including Linked Lists, dynamic memory allocation (`malloc`/`free`), pointer dereferencing, and pass-by-reference logic with structs.

## What I Mastered Today:
* **Linked List Foundation:** Understanding node architectures (`struct Node`) and linking sequential memory blocks scattered in RAM.
* **Pointer Traversal & Insertion:** Iterating through linked lists via pointers and prepending dynamic nodes to the head.
* **Dynamic Memory Management:** Reserving memory at runtime using `malloc()` and safely freeing memory blocks to prevent memory leaks.
* **Struct Pointers & Pass-by-Reference:** Passing structure pointers (`struct Student*`) into functions to directly modify underlying memory addresses (`s->marks`).
